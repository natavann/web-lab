### Dark Mode Toggle
- **Description**: Added a Call-to-Action (CTA) button in the header.
- **Functionality**: Uses JavaScript to toggle a `data-theme` attribute on the `body` tag.
- **Styling**: CSS selectors target `[data-theme="dark"]` to invert background and text colors for a comfortable night-time viewing experience.
